Objectives & Scope

Detect license plates from static and live images; extract readable alphanumeric content; support Indian and foreign formats; and display real-time recognition in a user-friendly web interface.

Implementation

The final implementation integrates SSD MobileNet for detection, EasyOCR for character recognition, and Flask for web interfacing. Users can upload images or activate a live camera feed to run detections, which are rendered dynamically on the website.

Challenges & Solutions

  • OCR issues on low-quality images resolved using EasyOCR
  • Live detection and web sync handled using Python threading
  • Foreign plate formats supported by relaxing character filters

Conclusion

This project demonstrated an end-to-end real-time license plate detection system with web access and camera support. It reflects strong technical understanding and readiness for deployment in practical applications.
